How to make an API call?

API call:

http://host/api/v1.0/price_today
http://host/api/v1.0/price_today/-id:number-
http://host/api/v1.0/price_today/update/-id:number-
http://host/api/v1.0/price_today/delete/-id:number-
http://host/api/v1.0/price_today/fuel/list_fuel
http://host/api/v1.0/price_today/region/list_region
http://host/api/v1.0/price_today/operator/list_operator

Parameter --id-- is an integer. This is ID of the one record. It provides data on the price of the fuel of a certain type, at certain gas station, on certain date, and in a certain area. If you have an access token, the record can be edited or deleted.
Here you need to specify the search region. The region is written in Ukraine. You can check the spelling on the website form for correct spelling.
Parameter --fuel_operator--. Write the name of the gas station as indicated on their official website.
Parameter --fuel--. Write the type of fuel. For gasoline A95 or A92, for diesel - diesel, for gas - gas. This URL will turn you a JSON file with fuel prices in the area. Additional query parameter --fuel-- used to filter by fuel type.
For example:

http://127.0.0.1:8000/fuel/region/Сумська

Get the price of fuel in Sumy region

http://127.0.0.1:8000/fuel/region/Сумська?fuel=A95

Get the price of fuel in Sumy region only for A95 fuel

How to get price history?

http://host/fuel/history/--region--or--fuel_operator--/--name region--or--name operator--?start_data=yyyy-mm-dd&end_data=yyyy-mm-dd

By default you can view the history for the last 2 days. To get history for a period you need to use the query parameter: --start_data-- and --end_data--. Parameter --region--. Here you need to specify the search region. The region is written in Ukraine. You can check the spelling on the website form for correct spelling.
Parameter --fuel_operator--. Write the name of the gas station as indicated on their official website.
Parameter --fuel--. Write the type of fuel. For gasoline A95 or A92, for diesel - diesel, for gas - gas.

For example:

http://127.0.0.1:8000/fuel/history/region/Сумська

Get prices for the last 2 days for the region Sumy

http://127.0.0.1:8000/fuel/history/fuel_operator/WOG?start_data=2022-11-16

In this example, we get data on WOG fuel station in th period from 2022-11-16 to today.

http://127.0.0.1:8000/fuel/history/region/Вінницька?start_data=2022-11-15&end_data=2022-11-17

Get prices in the period from 2022-11-15 to 2022-11-17 for the region Vinnitsiy

http://127.0.0.1:8000/fuel/history/region/львівська?start_data=2022-11-10&end_data=2022-11-30&fuel=Gas

Get prices in the period from 2022-11-10 to 2022-11-30 for the region Lviv
